# better-update

Self-hosted OTA update platform for Expo and React Native apps.

## Features

- OTA updates with staged rollouts, rollback, and channel promotion
- Branches and channels for routing devices to releases
- Native build pipeline (iOS, Android) with credential management
- Fingerprint-based compatibility matrix between builds and updates
- Per-project env vars, audit log, analytics, scoped API keys
- Multi-org teams, GitHub OAuth or email/password sign-in
